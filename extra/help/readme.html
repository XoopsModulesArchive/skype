This block of Xoops allows to access to the options of the users that use the programa Skype, to makes telephones calls through internet, voice over IP.

For it, off course you must have installed the programa ,that you can get from the download site of Skype.

1 - Install the block.
You follow this steps:

  • Copy the content in the folder /modules/, (/extra/ is not necessary to works the block), so, this is the structure
    /modules/index.php and xoops_version.php: general definitions of the block
    /modules/images/*.png: images
    /modules/blocks/skype.php: php code to create the block
    /modulate / language /: configured languages
    /extra/ this help, images and other files with other forms of seeing the block, to customize it. But it is not necessary that you copy this folder to works the block.
  • In Administration - > Modules, install it.
  • In Administration - > Blocks make it visible in the area that interests you.
  • In Administration - > Groups configure access to the user groups that you believe convenient.

2 - to customize the block:

There are 3 versions of the block. You will see them in /skype/extras/blocks/. To choose one of them, copy the corresponding file, renaming it to skype.php in skype/blocks/skype.php. Default that comes installed is skype1.php. This versions are

  • skipe1.php  Appears an image and text to call, a drop-down list with the other options and another link under toward the page of download of Skype
 Call me!
Download Skype
  • skipe2.php: It is a drop-down menu of images, with a link in botton of the page to download Skype. It is more beautiful, but it takes more slowly when is loading.

  • skipe3: In this case, appears visible all the options directly in links.
  • Call me!
  • My profile
  • Add me to skype
  • Start chat?
  • My Voicemail
  • Send me to File
  • Download Skype

3 - Add your name of contact of Skype.

In each file skipe1-2-3 php searches the lines where appears:

/***** Here your user yam in Skype *****/
$skype_user = "name_skype_user";

and change it in the way that, if your name is "johnsmith", it must be this way... and be careful about respect the quotation marks and the semicolon  at the end!

/***** Here your user yam in Skype *****/
$skype_user = "johnsmith";
